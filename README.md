# Hospital Data Analysis Dashboard

## Project Overview
This project analyzes hospital operational data using Microsoft Excel to identify trends in patient admissions, department performance, and hospital resource utilization.

## Problem Statement
Hospitals generate large amounts of data related to patient visits, treatments, and departments. This project transforms raw data into an interactive dashboard to support data-driven decision making.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning
- Dashboard Design

## Dashboard Features
- Total patient overview
- Department wise analysis
- Monthly patient trends
- Hospital performance metrics

## Key Insights
- Certain departments handle the majority of patient visits.
- Patient admissions vary across months.
- Dashboard visualization helps management quickly identify trends.
